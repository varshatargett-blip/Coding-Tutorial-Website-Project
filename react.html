React Introduction

React is a JavaScript library for building user interfaces. It is used to build single-page applications and allows us to create reusable UI components.

Last updated: 27 March 2026  ·  5 min read  ·  Topic 1 of 45

What is React?

  • React, sometimes referred to as a frontend framework, is a JavaScript library created by Facebook.
  • React is a tool for building UI components.
  • Instead of manipulating the browser's DOM directly, React creates a Virtual DOM in memory, where it does all the necessary manipulating, before making the changes in the browser DOM.
💡
Library vs Framework

React is technically a library, not a framework, because it is only concerned with rendering the UI. For full framework features (like routing), React relies on other packages like React Router.

Why use React?

React allows developers to create large web applications that can change data, without reloading the page. The main purpose of React is to be fast, scalable, and simple.

One of the biggest advantages of React is the ability to write components. Components are like independent, reusable Lego blocks that contain both structure (HTML) and logic (JavaScript).

React Syntax (JSX)

React uses JSX (JavaScript XML). JSX allows us to write HTML directly inside our JavaScript code.

Syntax Example
import React from 'react';
import ReactDOM from 'react-dom/client';

function Hello() {
  return <h1>Hello World!</h1>;
}

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<Hello />);
