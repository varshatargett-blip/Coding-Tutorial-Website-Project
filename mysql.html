MySQL Introduction

MySQL is the most popular Open Source Relational SQL Database Management System. It is developed, distributed, and supported by Oracle Corporation.

Last updated: 27 March 2026  ·  5 min read  ·  Topic 1 of 38

What is MySQL?

  • MySQL is a relational database management system (RDBMS).
  • MySQL is open-source (free to download and use).
  • MySQL is ideal for both small and large applications.
  • MySQL is very fast, reliable, scalable, and easy to use.
  • MySQL is cross-platform and compiles on a number of platforms.
💡
SQL vs MySQL: What's the difference?

SQL (Structured Query Language) is the standard language used to query and operate database systems. MySQL is the actual software program (the database management system) that stores the data and interprets the SQL language.

Why use MySQL?

MySQL is used by many of the largest websites in the world, including Facebook, Twitter, AirBnb, Booking.com, Uber, GitHub, and YouTube.

It is frequently used alongside PHP or Node.js to create dynamic websites. The data in a MySQL database are stored in objects called tables. A table is a collection of related data entries and it consists of columns and rows.

MySQL Query Syntax

Because MySQL uses standard SQL, the syntax for retrieving data looks exactly like standard SQL.

Syntax Example
SELECT username, email
FROM users
WHERE status = 'active';
