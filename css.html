CSS Introduction

CSS is the language we use to style an HTML document. It describes how HTML elements should be displayed on screen, paper, or in other media. This tutorial will teach you CSS from basic to advanced.

Last updated: 27 March 2026  ·  6 min read  ·  Topic 1 of 40

What is CSS?

CSS stands for Cascading Style Sheets. If HTML is the skeleton of a webpage, CSS is the skin, clothing, and makeup. It dictates colors, fonts, layouts, and animations.

  • CSS describes how HTML elements are to be displayed on screen.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.
💡
Why is it called "Cascading"?

The term "cascading" means that styles can fall (or cascade) from one stylesheet to another, enabling multiple stylesheets to be used on one HTML document. The last rule applied wins!

CSS Syntax

A CSS rule consists of a selector and a declaration block.

Syntax Example
h1 {
  color: blue;
  font-size: 12px;
}
TermDescription
SelectorPoints to the HTML element you want to style (e.g., h1, p, .class).
DeclarationContains a property name and a value, separated by a colon.
PropertyThe specific style you want to change (e.g., color, margin).
ValueThe outcome you want for the property (e.g., blue, 10px).

Why Use CSS?

Before CSS, tags like <font>, and color attributes were used directly inside HTML. This was a nightmare for developers! If you wanted to change the font color of every heading on a large website, you had to edit every single HTML file manually.

CSS solved this problem by removing the style formatting from the HTML page. Now, you can change the look of an entire website by changing just one file!
