SQL Introduction

SQL is a standard language for storing, manipulating (हेरफेर - handling/changing), and retrieving data in databases. This tutorial will teach you SQL from basic to advanced.

Last updated: 27 March 2026  ·  5 min read  ·  Topic 1 of 38

What is SQL?

  • SQL stands for Structured Query Language.
  • SQL lets you access and manipulate databases.
  • SQL became a standard of the American National Standards Institute (ANSI) in 1986, and of the International Organization for Standardization (ISO) in 1987.
💡
History Fact

SQL was initially developed at IBM by Donald D. Chamberlin and Raymond F. Boyce in the early 1970s. It was originally called SEQUEL (Structured English Query Language).

What Can SQL Do?

SQL is a declarative (घोषणात्मक - declarative/announcing) language, which means you tell the system what to do, not how to do it. It is used to perform CRUD (Create, Read, Update, Delete) operations on relational (संबंधपरक - relational) databases like MySQL, SQL Server, Oracle, and PostgreSQL.

  • SQL can execute queries against a database.
  • SQL can retrieve data from a database.
  • SQL can insert records in a database.
  • SQL can update records in a database.
  • SQL can delete records from a database.

SQL Syntax

A SQL statement consists of keywords (like SELECT, FROM, WHERE) followed by the specific table or column names. It is important to know that SQL keywords are NOT case sensitive (select is the same as SELECT), but writing them in uppercase is a common industry practice.

Syntax Example
SELECT *
FROM Customers;

Semicolon after SQL Statements?

Some database systems require a semicolon at the end of each SQL statement. Semicolon is the standard way to separate each SQL statement in database systems that allow more than one SQL statement to be executed in the same call to the server.
